Setup:
  python -m venv venv                    Create virtual environment

Data:
  python generate_sample_data.py         Generate test data
  python verify_dataset.py               Check dataset

Training:
  cd src                                 Go to source folder
  python train_model.py                  Train ML models
  cd ..                                  Return to root

Testing:
  python test_system.py                  Run all tests
  python quick_start.py                  Show setup status

Running:
  python app.py                          Start web server
  
API Key:
  set OPENWEATHER_API_KEY=your_key       Set API key (Windows)

┌──────────────────────────────────────────────────────────────────────────────┐
│  🐛 TROUBLESHOOTING                                                          │
└──────────────────────────────────────────────────────────────────────────────┘

Problem: Module not found
Solution: venv\Scripts\activate && pip install -r requirements.txt

Problem: Dataset not found
Solution: python generate_sample_data.py

Problem: Model not loaded
Solution: cd src && python train_model.py && cd ..

Problem: Weather API error
Solution: Check API key: echo %OPENWEATHER_API_KEY%

Problem: Port 5000 in use
Solution: Edit app.py, change port to 5001

Problem: Any other issue
Solution: python test_system.py (shows what's wrong)
